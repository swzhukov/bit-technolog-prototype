## Сквозной сценарий (5 минут)

1. **Откройте** `http://217.114.7.5:8081/dashboard` — Мои задачи (4-5 задач)
2. **Кликните «Открыть»** на Упор продольный → карточка детали
3. **«Чертёж + Операции»** — слева чертёж, справа 6 операций со светофором
4. **«Почему такая норма?»** — обоснование (факт / аналог / AI)
5. **«Ресурсная спецификация»** — предпросмотр РС + отчёт трансформации
6. **«⬇ Выгрузить в 1С:ERP»** — файл XML
7. **«📬 Извещения»** → И-2026-014 — diff «было/стало»
8. **«Принять правки»** → новая версия ТК → перегенерация РС
