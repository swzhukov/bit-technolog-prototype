Создание новой детали

В текущей версии БИТ.Технолог (v0.8) создание новых изделий и деталей выполняется администратором через загрузку спецификации из 1С или через прямое добавление в базу данных.

В пилотной версии для участников из Техинкома доступно 51 предзаполненное изделие (КАМАЗ-43118, УРАЛ-4320 и др.) с примерами техкарт. Этого достаточно, чтобы оценить работу системы на реальных данных.

Когда появится полноценная форма?

В версии после пилота (Sprint 6, ориентировочно август 2026) будет реализована:

  • Форма создания изделия с полями: обозначение, наименование, тип, цех, материал
  • Загрузка спецификации из Excel/1С
  • Привязка к чертежу (PDF/DWG)
  • Валидация по справочникам (профессии, оборудование)

До этого момента: обратитесь к администратору системы, чтобы добавить новое изделие.
