Что это?

Прототип AI-помощника технолога. По свойствам детали (от конструктора из КОМПАС-3D) генерирует черновик техкарты:

  • Список операций с оборудованием и временем
  • Указание источника оценки (ГОСТ / аналог / правило)
  • Уверенность для каждой операции
  • Warnings с цитатами из свойств детали
  • Вопросы к технологу с вариантами ответа
  • Обоснование решений (для руководства)

Кликните на любую деталь, чтобы попробовать.
